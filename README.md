# 🏥 HealthVault – Digital Health Record Management System

HealthVault is a full-stack web application that enables patients and healthcare providers to securely manage digital health records. The application includes user authentication, role-based access, medical record management, QR-based sharing, and an AI-powered health chatbot.

## 🚀 Live Demo

**Frontend:**
https://tejashribambal19.github.io/healthvault/

**Backend API:**
https://healthvault-backend-3f84.onrender.com

---

## ✨ Features

* 🔐 Secure user authentication with JWT
* 👤 Role-based access (Patient, Doctor, Admin)
* 📄 Upload and manage medical records
* 📤 Share records securely using QR codes
* 🤖 AI-powered health chatbot
* 📱 Responsive user interface
* ☁️ Cloud deployment with GitHub Pages, Render, and Supabase

## ⚙️ Installation

### Clone the repository

```bash
git clone https://github.com/Tejashribambal19/healthvault.git
cd healthvault
```

### Backend

```bash
cd backend
npm install
cp .env.example .env
npm run dev
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```
